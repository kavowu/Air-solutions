台灣空調工程的系統性問題檢討

從設計、施工、法規、運維四個維度，分析空調系統的結構性缺陷

⚠️

問題 1：顯熱負荷優先，除濕能力不足

台灣夏季外氣條件：乾球溫度 33–35℃，露點溫度 24–27℃，年平均相對濕度約 80%。

傳統空調設計以顯熱負荷（溫度）為核心，潛熱負荷（濕度）被邊緣化。結果是：

  • • 冷卻能力強，但除濕效率低
  • • 室內相對濕度難以控制在 50–60% 的舒適範圍
  • • 使用者感覺悶熱，傾向進一步降溫
⚠️

問題 2：缺乏獨立新風處理

現代建築為追求能效，設計得極其密閉。但空調系統往往缺乏獨立的新風處理單元。

結果是：

  • • 新風量不足或完全依賴滲漏
  • • 室內 CO₂、VOCs、異味累積
  • • 使用者錯誤地將悶熱歸因於溫度
  • • 進一步降溫，加劇結露風險
⚠️

問題 3：出風口與回風口設計不良

許多系統的出風口設計未考慮結露風險：

  • • 出風口表面溫度過低（低於室內露點）
  • • 風速分布不均，局部冷點結露
  • • 回風口位置不當，導致短路循環
  • • 冷凝水無法及時排出
⚠️

問題 4：材料選用未考慮防結露

出風口、風管、冷凝水盤等材料選用往往忽視防結露需求：

  • • 金屬材料導熱快，易結露
  • • 缺乏防結露保溫層
  • • 冷凝水盤無保溫，水溫過低
  • • 材料表面粗糙，易滋生黴菌

設計改善建議

✓

獨立新風系統

配置新風處理單元，確保新風量 ≥ 6 ACH（每小時換氣次數）

✓

除濕優先設計

將室內相對濕度控制在 50–60% 作為核心設計目標

✓

防結露出風口

出風口表面溫度 ≥ 室內露點 + 2℃（安全餘裕）

✓

材料與保溫

選用防結露材料，風管與冷凝水盤加保溫層

⚠️

問題 1：風管表面灰塵堆積

風管內部長期暴露於空氣中，灰塵、污垢、黴菌孢子不斷沉積。

後果：

  • • 風阻增加，風量下降
  • • 黴菌滋生，釋放孢子
  • • 異味產生，空氣品質惡化
  • • 能耗增加 15–30%
⚠️

問題 2：冷凝水盤缺乏保溫

冷凝水盤直接暴露在冷空氣中，表面溫度可能低於 10℃。

結果是：

  • • 冷凝水結冰，堵塞排水
  • • 盤內積水，黴菌滋生
  • • 異味散發，污染室內空氣
  • • 漏水損害天花板和隔音棉
⚠️

問題 3：濾網未定期更換

許多業主對濾網維護認識不足，導致濾網長期不換。

危害：

  • • 濾網堵塞，風量大幅下降
  • • 黴菌在濾網上大量繁殖
  • • 冷卻效率下降，能耗增加
  • • 室內空氣品質惡化
⚠️

問題 4：漏水與天花板發霉

冷凝水排水不暢或漏水，導致天花板隔音棉受潮。

結果是：

  • • 隔音棉吸水，黴菌大量滋生
  • • 黴菌孢子進入室內空氣
  • • 異味嚴重，影響健康
  • • 維修成本高，可能需要更換天花板

施工與維護改善建議

✓

定期清潔風管

每年至少清潔一次，使用專業風管清潔設備

✓

冷凝水盤保溫

加裝保溫層，確保盤內溫度 ≥ 15℃

✓

濾網維護計畫

建立定期更換計畫（通常 1–3 個月）

✓

排水系統檢查

定期檢查冷凝水排水暢通，防止積水

台灣空調驗收標準的核心問題

現狀：溫度中心主義

現行驗收標準（如 CNS、ASHRAE）以室內溫度作為核心指標，濕度、新風量、空氣品質被邊緣化。

典型驗收標準：

  • • 室內溫度：22–26℃（✓ 易達成）
  • • 相對濕度：40–70% RH（✗ 很難達成）
  • • 新風量：未明確規定或被忽視
  • • IAQ 指標：不在驗收範圍內

後果：設計與施工的偏差

  • • 設計師優先確保溫度達標，忽視濕度控制
  • • 施工單位只關注溫度驗收，不投入除濕設計
  • • 業主驗收通過後，系統實際性能與預期不符
  • • 後期維修成本高，責任難以界定

法規改善建議

✓

修訂驗收標準

將濕度控制（40–60% RH）、新風量（≥6 ACH）、IAQ 指標納入驗收範圍

✓

強制 IAQ 設計

新建築設計必須納入 IAQ 考量，並在設計階段明確規定

✓

建立污染物控制標準

明確規定室內 CO₂、VOCs、PM2.5、黴菌孢子等污染物的上限

⚠️

問題 1：設定溫度過低

許多使用者將空調設定在 22–24℃，甚至更低。

原因與後果：

  • • 原因：污染物累積導致的悶熱感被誤認為溫度不足
  • • 結果：出風口結露，冷凝水盤積水
  • • 後果：黴菌滋生，異味加重，進一步降溫
  • • 能耗：每降低 1℃，能耗增加 5–10%
⚠️

問題 2：缺乏定期維護計畫

許多建築物缺乏系統的 AC 維護計畫。

結果是：

  • • 濾網長期不換，風量下降
  • • 風管積塵，黴菌滋生
  • • 冷凝水盤積水，異味散發
  • • 系統性能逐年下降，無法察覺
⚠️

問題 3：缺乏 IAQ 監測

大多數建築物沒有 IAQ 監測設備。

後果：

  • • 無法量化污染物濃度
  • • 無法判斷系統是否達標
  • • 無法及時發現問題
  • • 業主與承包商責任難以界定
⚠️

問題 4：人為操作不當

使用者對空調系統的特性認識不足。

常見錯誤：

  • • 關閉新風進口，導致空氣品質惡化
  • • 長期不開窗，污染物累積
  • • 忽視濾網更換，系統性能下降
  • • 過度降溫，導致結露和黴菌

運維改善建議

✓

設定溫度指南

夏季建議設定 26–28℃，冬季 20–22℃，避免過度降溫

✓

定期維護計畫

建立濾網更換、風管清潔、冷凝水盤檢查的定期計畫

✓

IAQ 監測系統

安裝 CO₂、濕度、PM2.5 等傳感器，實時監測空氣品質

✓

使用者教育

提供操作指南，幫助使用者理解 AC 系統的最佳實踐

核心結論

台灣空調系統的根本問題

空調系統設計、施工、法規、運維四個環節均存在結構性缺陷，導致防霉、去異味、節能三大難題無法根本解決。

傳統的「降溫」思路只會加劇結露和黴菌問題，形成惡性循環。

PICT 技術的必要性

PICT 技術通過主動分解污染物、抑制微生物、消除悶熱感，打破惡性循環。

它不是替代空調，而是補足空調系統的功能缺口，實現真正的舒適、健康、節能。
